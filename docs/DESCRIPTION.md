# Cadence — Short Description

## Primary (326 characters)

> Cadence is a content calendar built for social media managers and agencies running
> many client accounts at once. Plan, approve, and publish every brand's month from one
> board — with per-channel post variants, drag-and-drop scheduling, an AI idea
> generator, and shareable approval links your clients can use without an account.

*Use this as the default: app store listing, landing page subhead, README intro.*

## Notes on wording

- **"many client accounts"** is doing the important work. It signals immediately that
  this is not another single-brand scheduler, which is the main thing that
  differentiates Cadence.
- **"without an account" / "no login"** is the strongest single selling point for the
  approval flow and worth keeping in any version that has room.
- The primary uses British *"colour"* only in the alternates; swap to *"color"*
  throughout for US-facing copy.
